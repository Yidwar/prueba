Ejercicio SQL

Supongamos que tienes tres tablas en una base de datos: clientes, pedidos, y productos. Diseña una consulta SQL que muestre el nombre de los clientes junto con los nombres de los productos que han pedido en los últimos 30 días. Clientes: id_cliente, nombre Productos: id_producto, nombre_producto, precio_unitario Pedidos: id_pedido, id_producto, cantidad, precio_unitario,

Table clientes:

                CREATE TABLE clientes (
                id_cliente INT AUTO_INCREMENT PRIMARY KEY,
                nombre VARCHAR(255)
                );

Table productos:

                CREATE TABLE productos (
                id_producto INT AUTO_INCREMENT PRIMARY KEY,
                nombre_producto VARCHAR(255),
                precio_unitario DECIMAL(10, 2)
                );

Table pedidos:

                CREATE TABLE pedidos (
                id_pedido INT AUTO_INCREMENT PRIMARY KEY,
                id_cliente INT,
                id_producto INT,
                cantidad INT,
                fecha_pedido DATE,
                FOREIGN KEY (id_cliente) REFERENCES clientes(id_cliente),
                FOREIGN KEY (id_producto) REFERENCES productos(id_producto)
                );

solución:

                SELECT c.nombre AS NombreCliente, pr.nombre_producto AS NombreProducto
                FROM clientes c
                JOIN pedidos pe ON c.id_cliente = pe.id_cliente
                JOIN productos pr ON pe.id_producto = pr.id_producto
                WHERE pe.fecha_pedido >= DATE_SUB(CURDATE(), INTERVAL 30 DAY);
